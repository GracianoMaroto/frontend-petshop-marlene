# Frontend PetShop DM

Interface em React + Vite para consumir a API do backend PetShop DM.

## Rodando localmente

```bash
npm install
npm run dev
```

Por padrão, o frontend consome a API em `http://localhost:3000`.

Para trocar a URL, crie um `.env` no frontend:

```env
VITE_API_URL=http://localhost:3000
```
